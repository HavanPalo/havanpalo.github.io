Datamaskinens oppbygging og virkemåte

Prosessor

Intel

Amerikans produsent av prosessorer Grunnlagt 1968 - laget intel 4004 CPU, 4 bits cpu Dagens CPU er 64 bits. 4 bit gir 2^4 som største tall = 16. Dagens CPU kan jobbe med tall på på 2^64 = 18446744073709551616.

AMD

AMD (Advanced Micro Devices) grunnlagt i 1969 Samme år lanserte de sin egen klone av 8-bit prosessoren Intel 8080, og kalte den AM9080ADC

ARM

Acorn (advanced) RISC (reduced Instruction Set Computer) Machine. RISC CPU-er har få og enkle instruksjoner (i motsetning til Intel) Hver instruksjon skla ta minst mulig tid. Kompliserte beregninger består av mange instruksjoner. Brukes i mobiler

Prosessoren er hjernen i datamaskinen Hastigheten på en CPU er mellom 2-4 GHz for nye maskiner 1ns er 2.7m

Ram

RAM er en forkortelse som betyr Random Access Memory

Lagring

Standard lagringsmedia i dag er SSD (Solid State Drive) Tidligere var HDD (Hard Disk Drive) det mest brukte
